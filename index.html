HTML

HTML son las siglas Hypertext Markup Language, que puede traducirse como lenguaje de marcas o marcado de hipertexto.

El lenguaje HTML se emplea para crear las paginas, Es muy facil ver el codigo de HTML de una pagina web, la opcion exacta cambia de un navegador a otr y tambien puede cambiar de una version a otra de un mismo navegador, pero suelen tener un nombre similar. HTML se compone de equitas que se escriben entre los simbolos menor que y mayor que.
